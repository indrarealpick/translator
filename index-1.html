⚙️ Konfigurasi API
Dapatkan API key gratis di openrouter.ai/keys. API key disimpan di localStorage browser Anda.
Default: https://openrouter.ai/api/v1/chat/completions
Contoh: openai/gpt-4o-mini, google/gemini-flash-1.5, meta-llama/llama-3.1-8b-instruct:free
⚠️ Perhatian Keamanan: Menyimpan API key di frontend tidak aman untuk produksi. Gunakan Cloudflare Worker / backend proxy untuk menyembunyikan API key dari publik.
